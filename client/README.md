## ‚ú® Features

- **Real-Time Messaging:** Instant message delivery and reception using WebSockets (Socket.io).
- **Secure Authentication:** Robust user registration and login system with JWT (JSON Web Tokens) and bcrypt password hashing.
- **Modern UI/UX:** Clean, responsive, and intuitive user interface built with Tailwind CSS and Radix UI components. Supports both light and dark modes.
- **State Management:** Efficient client-side state management using Zustand.
- **RESTful API:** Well-structured backend API using Node.js and Express.
- **Cloud Database:** MongoDB Atlas integration for secure and scalable data storage.
